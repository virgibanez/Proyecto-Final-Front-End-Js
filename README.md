**Propósito""
Este proyecto es una página web para un emprendimiento de repostería artesanal.
El sitio incluye un catálogo de productos, un carrusel de reseñas de clientes y un formulario para un club de beneficios.
El carrito de compras permite ver el detalle de la compra en una ventana emergente y enviar el pedido de forma automática a WhatsApp para coordinar el pago.

**Tecnologías usadas:"
- HTML5: Estructura de las secciones, el catálogo y el formulario.
- CSS3: Diseño visual, adaptación para celulares y el movimiento del carrusel de reseñas.
- JavaScript: Lógica del carrito de compras, gestión del almacenamiento local (LocalStorage) y manejo de la ventana emergente.
- Google Fonts: Tipografías para los títulos y párrafos.
- Formspree: Funcionamiento del formulario de contacto.
